🪶 Meet Wren

Wren is a game engine that doesn't fight you.
It’s fast, modern, and puts creative control back where it belongs — in your hands.

  • 🚀 Built with Rust — performance and safety by default
  • 🔩 Explicit architecture — no hidden magic, no guesswork
  • 🎮 2D first, with clean pathways to 3D support
  • 🖱️ Visual scripting (soon) — prototype without writing Rust
  • 💾 Tiny footprint — forget 100GB installs
  • 🧩 Modular + extensible — add only what you need, use any crate

Programmers. Artists. Designers. Beginners. Wren is for all of you.
It’s a tool that helps you build — not one you have to wrestle with.
