- 🔍 Integrated Clippy + formatting in build script
- 📂 Modular architecture — core first, editor later
- 🐛 Debug utilities (WIP)
- 🏗️ Planned editor separation (Wren Studio)

- 🧱 ECS system (custom or external crate)

- 🎮 Input abstraction
- 📜 Scene management
